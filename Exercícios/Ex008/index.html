Principais Formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito. Usando o tag <b> (não semântica)

Nesta frase, temos um termo em destaque. Usando a tag <strong> (com semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico. Usando a tag <i>(não semântica)

Nesta frase, temos um termo em ênfase. Usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno. Com a tag <big> (Já Obsoleta mas usavel) e <small> (Ainda Funcional)

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. Com a tag <del>

Texto inserido

Podemos marcar um texto como inserido pra indicar que ele foi adicionado depois. Com a tag <ins> ou a tag <u>(Para Não semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20 + 3. Com a tag <sup>

Texto Subscrito

Para Inserir coisas do tipo h2o. Com a tag <sub>
